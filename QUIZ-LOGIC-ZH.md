# OneCo 测试 — 类型与逻辑详解

## 5 种人格类型

每种人格代表一种"是否适合做一人公司"的性格原型：

| 代码 | 名称 | 图标 | 核心特质 |
|------|------|------|----------|
| `visionary` | 远见者 | 🔭 | 思考可能性、发现规律、在空白画布前充满能量 |
| `executor` | 执行者 | ⚡ | 行动派、持续交付、把混乱变成系统 |
| `connector` | 连接者 | 🌐 | 快速建立信任、天生的社群构建者、深谙人的动机 |
| `expert` | 专家 | 🎯 | 深耕而非泛学、成为别人求助的对象、深度即差异化 |
| `creative` | 创意人 | 🎨 | 通过作品表达、让人产生共鸣、设计驱动型思维 |


## 计分逻辑

### 人格得分（你是哪种类型？）

共 10 道题，每道题 4 个选项（A/B/C/D），每个选项标记了对应的人格类型。

用户完成测试后，统计每种类型被选中的次数：

```
visionary: 0, executor: 0, connector: 0, expert: 0, creative: 0
→ 每选一个答案，对应类型 +1
→ 最高分的类型 = 你的人格类型
```

单个类型最高分为 10（如果每道题都选了同一类型），最低为 0。

**平局处理：** 如果两个类型分数相同，按对象定义顺序排列（`visionary > executor > connector > expert > creative`），排在前面的胜出。这是因为 `Object.entries().sort()` 是稳定排序，相同值保持原始顺序。


### 创业准备度（你该不该单干？）

这是一个独立的分数（0–4），衡量你有多"适合创业"，与人格类型无关。

只有 **4 道特定题目** 计入此分数：第1、5、7、8题（0索引：0, 4, 6, 7）。

每道题只有 **1 个特定答案** 给创业分（标记 `isStartupPoint: true`）：

| 题目 | 给分答案 | 对应人格 |
|------|---------|----------|
| Q1："当你有一个新想法时..." | C："立刻开始动手做" | executor |
| Q5："面对不确定性..." | A："兴奋——更多可能性！" | visionary |
| Q7："你对金钱和风险的态度..." | A："好工作自然会带来金钱" | creative |
| Q8："你在什么状态下工作最高效？" | A："由你自己制定规则" | executor |

创业准备度 = 你选中了上面几个答案（0 到 4）。

结果页用"音量条"可视化展示：

| 分数 | 标签 | 含义 |
|------|------|------|
| 3–4 | "一人公司材料" | 全速前进，适合单干 |
| 1–2 | "先搞副业现金流" | 混合模式，边打工边创业 |
| 0 | "先打好基础" | 先工作积累，再考虑单干 |


## 每道题的完整映射

### Q1："当你有一个新想法时，你会..."
- A：先研究所有信息 → **expert（专家）**
- B：马上告诉所有人 → **connector（连接者）**
- C：立刻开始动手做 → **executor（执行者）** ⭐ 创业加分
- D：写一份详细计划 → **visionary（远见者）**

### Q2："最让你消耗精力的事情是..."
- A：重复性工作 → **visionary（远见者）**
- B：独自工作 → **connector（连接者）**
- C：太多规则和限制 → **executor（执行者）**
- D：目标不清晰 → **expert（专家）**

### Q3："你通常如何做决定？"
- A：直觉 → **creative（创意人）**
- B：数据和研究 → **expert（专家）**
- C：询问别人的意见 → **connector（连接者）**
- D：慢慢权衡利弊 → **visionary（远见者）**

### Q4："在团队项目中，你自然而然地成为..."
- A：出主意的那个人 → **visionary（远见者）**
- B：负责组织协调的人 → **expert（专家）**
- C：真正动手执行的人 → **executor（执行者）**
- D：维系团队关系的人 → **connector（连接者）**

### Q5："面对不确定性，你会感到..."
- A：兴奋——更多可能性！ → **visionary（远见者）** ⭐ 创业加分
- B：焦虑，但我会坚持 → **executor（执行者）**
- C：我需要尽快消除不确定性 → **expert（专家）**
- D：等有更多信息才能行动 → **creative（创意人）**

### Q6："你最自豪的工作是..."
- A：改变了人们的思维方式 → **visionary（远见者）**
- B：直接帮助了某人 → **connector（连接者）**
- C：按时完成交付 → **executor（执行者）**
- D：工艺精美，令人赏心悦目 → **creative（创意人）**

### Q7："你对金钱和风险的态度是..."
- A：好工作自然会带来金钱 → **creative（创意人）** ⭐ 创业加分
- B：先有稳定再冒险 → **expert（专家）**
- C：只接受经过计算的风险 → **visionary（远见者）**
- D：如果我相信，我愿意大胆押注 → **executor（执行者）**

### Q8："你在什么状态下工作最高效？"
- A：由你自己制定规则 → **executor（执行者）** ⭐ 创业加分
- B：有明确的结构和流程 → **expert（专家）**
- C：与他人合作 → **connector（连接者）**
- D：有大块的专注时间 → **creative（创意人）**

### Q9："你的超能力是..."
- A：发现别人忽略的规律 → **visionary（远见者）**
- B：连接人与人 → **connector（连接者）**
- C：让事情真正发生 → **executor（执行者）**
- D：让事物变得美好 → **creative（创意人）**

### Q10："5年后，你希望自己感到..."
- A：自由 → **executor（执行者）**
- B：有影响力 → **connector（连接者）**
- C：受人尊重 → **expert（专家）**
- D：充满创意 → **creative（创意人）**


## 各类型在题目中的出现频率

每种人格作为选项出现在 40 个答案中的次数：

| 人格 | 出现次数（共40个选项） | 占比 |
|------|----------------------|------|
| visionary（远见者） | 8 | 20% |
| executor（执行者） | 9 | 22.5% |
| connector（连接者） | 8 | 20% |
| expert（专家） | 8 | 20% |
| creative（创意人） | 7 | 17.5% |

**注意：** executor 有轻微优势（9次 vs 其他7-8次），统计上更容易得到"执行者"结果。creative 选项最少（7次），是最难获得的人格类型。


## 结果页逻辑

URL格式：`/result/{profileKey}?startup={score}`

每种人格展示两条路径：
1. **"如果你选择单干"** — 创业建议、商业模式、推荐角色
2. **"如果你先积累经验"** — 打工建议、目标岗位、选工作要看什么

加上创业准备度可视化（音量条动画）和 Jobpilot 引导 CTA。


## 数据追踪（Supabase）

用户完成测试后，以下数据保存到 `quiz_results` 表：

| 字段 | 说明 |
|------|------|
| `answers` | 可读格式，如 `{ "Q1": "C - 立刻开始动手做 (executor)" }` |
| `profile_type` | 最终人格类型 |
| `scores` | 5种人格分数 + 创业准备度 |
| `lang` | 用户使用的语言 |
| `referrer` | 来源页面（如小红书） |
| `utm_source/medium/campaign` | UTM追踪参数 |
| `device_type` | 移动端 / 平板 / 桌面端 |
| `screen_width` | 视口宽度 |
| `user_agent` | 浏览器信息 |
| `quiz_duration_seconds` | 完成测试用时 |
| `session_id` | 浏览器会话唯一ID |
